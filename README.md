# Portfolio

A modern, interactive professional portfolio showcasing projects, selected works, and technical certifications. Built with React and styled with Tailwind CSS, featuring smooth animations using GSAP and Framer Motion.

## 🚀 Features

- **Hero & Background:** Interactive, animated hero section with custom cursors.
- **Selected Works:** Dedicated showcases for diverse projects including KMIS, PDAO Dashboard, and Robotics.
- **Certifications:** Highlights continuous learning in areas like AI, Data Analytics, and Data Science.
- **Tools & Tech:** A visual overview of the tech stack and tools mastered.
- **Dark/Light Mode:** Full theming support via custom ThemeProvider.
- **Fluid Animations:** Powered by `framer-motion` and `gsap`.

## 🛠️ Tech Stack

- **Framework:** React 19 + Vite + TypeScript
- **Styling:** Tailwind CSS + Tailwind Merge
- **Animations:** GSAP, Framer Motion
- **Icons:** Lucide React
- **Routing:** React Router

## 📂 Project Structure

- `/src/components` - Core UI components (`Hero`, `SelectedWorks`, `Certifications`, `ToolsAndTech`).
- `/src/lib` - Utility functions (e.g., tailwind merge).
- `/public/assets` - Project screenshots and certification images.

## 🏃‍♂️ Getting Started

### Prerequisites
- Node.js (v18+ recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Suarenz/Portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## 📄 Scripts

- `npm run dev` - Starts the development server.
- `npm run build` - Builds the app for production.
- `npm run preview` - Locally preview the production build.
- `npm run lint` - Runs TypeScript type checking.
